# Mística-Linting
A simple assistant **for Sketch** to keep your files *FULL* Mística.

## Install
1. Copy and paste this line in your browser  
2. `sketch://add-assistant?url=https://registry.npmjs.org/mistica-linting/-/mistica-linting-1.1.3.tgz`
